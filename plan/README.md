# Plan

本目录用于记录项目整体目标和分阶段开发计划。

## 文件说明

- `overall-goals.md`
  - 项目整体目标与初步验收成果
- `phase1.md`
  - 项目方向与文档基础阶段
- `phase2.md`
  - MVP 系统骨架搭建阶段
- `phase3.md`
  - 结果展示增强与稳定化阶段
- `phase4.md`
  - 兼容优化模型与批量能力阶段
- `phase5.md`
  - 比赛展示优化与最终交付阶段

## 使用方式

建议开发时按阶段推进，并在每个阶段结束后检查：

- 阶段目标是否完成
- 阶段产出是否具备
- 阶段验收标准是否满足
