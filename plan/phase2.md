# Phase 2

## 阶段主题

MVP 系统骨架搭建

## 阶段目标

基于当前 `YOLOv8-seg` 模型，搭建前后端最小可运行闭环。

## 主要任务

- 搭建前端基础框架
- 搭建 `FastAPI` 后端基础框架
- 建立统一推理入口
- 建立模型注册与适配层
- 建立统一结果协议
- 完成单张图像上传与推理流程

## 阶段产出

- 可运行前端页面
- 可调用后端接口
- 当前 YOLOv8 模型接入完成
- 单张图识别闭环跑通

## 验收标准

- 用户可以上传单张图像
- 后端可调用当前 YOLOv8 模型
- 前端可展示病害类别、Mask、置信度、推理耗时
- 系统可导出标准 JSON 结果
